# Симуляция Доисторической Цивилизации

Браузерная симуляция развития ранней человеческой цивилизации, где доисторические люди образуют сообщества, собирают ресурсы, создают инструменты и развивают свои поселения.

## Особенности

- **Динамическое образование сообществ**: Люди автоматически формируют деревни при скоплении в группах
- **Управление ресурсами**: Сбор палок, древесины, камня и меди для поддержки роста населения
- **Система крафта инструментов**: Прогресс от деревянных инструментов → каменных → медных с повышением эффективности
- **Механика населения**:
  - Система размножения с генетическим наследованием
  - Лимиты населения, основанные на доступных инструментах и ресурсах
  - Система королевской линии для потомков лидеров
- **Визуальные индикаторы**:
  - Цветовое кодирование сообществ
  - Наложения статуса, показывающие текущие действия
  - Королевские ореолы для потомков лидеров
  - Детальные подсказки при выборе
- **Интерактивные элементы управления**:
  - Панорамирование и зум
  - Ручной спавн людей
  - Переименование поселений и жителей
  - Корректировка населения

## Как запустить

1. Клонируйте репозиторий
2. Убедитесь, что у вас установлены Node.js и npm
3. Установите зависимости:
   ```bash
   npm install
   ```
4. Запустите сервер разработки:
   ```bash
   npm run dev
   ```
5. Откройте браузер по адресу `http://localhost:5173` (или порту, показанному в терминале)

## Ключевые механики

### Атрибуты людей
- **Здоровье**: Определяет выживаемость
- **Сила**: Влияет на эффективность сбора ресурсов
- **Интеллект**: Влияет на скорость выполнения задач
- **Харизма**: Влияет на потенциал лидерства

### Прогрессия инструментов
1. **Деревянные инструменты**: Базовый сбор (требуется 4 древесины)
2. **Каменные инструменты**: На 60% быстрее сбор древесины (требуется 8 древесины + 4 камня)
3. **Медные инструменты**: На 80% быстрее сбор древесины + на 60% быстрее сбор камня (требуется 16 древесины + 4 меди)

### Лимиты населения
Лимит населения каждого поселения рассчитывается как:
```
Базовый лимит: 4
+ Деревянные инструменты: +1 за инструмент (макс. 10)
+ Каменные инструменты: +2 за инструмент (макс. 20)
+ Медные инструменты: +3 за инструмент (без явного максимума)
```

### Система пересадки деревьев
Женщины-жители с завершённым периодом восстановления после размножения автоматически сажают деревья во время отдыха, помогая поддерживать возобновляемые ресурсы.

## Управление

- **Клик по человеку**: Выделить/снять выделение для просмотра детальной подсказки
- **Клик по ресурсу**: Просмотр деталей ресурса при выделении
- **Перетаскивание фона**: Панорамирование вида
- **Колёсико мыши**: Зум_in/зум_out
- **Кнопка "Спавн человека"**: Добавление нового человека в центр
- **Кнопка "Перезапуск"**: Сброс симуляции с текущей настройкой населения
- **Кнопка "Сброс камеры"**: Возврат к виду по умолчанию
- **Иконки переименования (✎)**: Клик для переименования поселений или жителей
